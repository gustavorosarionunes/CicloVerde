# CicloVerde
Trabalho de Fabrica de Software

Meu nome é Gustavo e esse é o meu projeto desenvolvido para a disciplina de Fábrica de Software da Univille, pelo professor William Sestito. 

Este é o meu Projeto de Nivelamento. O foco aqui não foi criar o sistema mais complexo do mundo, mas sim provar que entendo a estrutura, a organização e a lógica por trás da criação de telas e navegação.

##  Sobre o Projeto
O CicloVerde foi pensado para resolver um problema muito comum nas nossas cidades: o descarte irregular de lixo e a falta de informação. Muita gente joga móveis e entulhos em terrenos baldios simplesmente porque não sabe como pedir para a prefeitura buscar ou onde é o lugar certo para descartar. 

O sistema simula um aplicativo voltado para o cidadão, onde é possível:
- **Procurar Resíduo:** Saber exatamente como embalar e onde jogar cada tipo de lixo.
- **Orientação de Descarte:** Ver os pontos de coleta (usei endereços reais de Joinville/SC para simular).
- **Agendar Retirada:** Pedir para recolherem coisas grandes (móveis, eletrodomésticos).
- **Fazer Denúncia:** Avisar sobre lixo em local proibido, com opção de mandar foto.
- **Gerar Protocolo:** Simulação de geração e acompanhamento de chamados.

## Tecnologias Utilizadas
Como o foco do projeto era o nivelamento e o entendimento real do código, eu decidi fazer tudo de forma "raiz", sem frameworks mágicos ou códigos copiados da internet que eu não saberia explicar. 

Usei apenas:
* **HTML5:** Para toda a estrutura, formulários e links entre as telas.
* **CSS3:** Para a estilização, cores, botões e para dar aquela cara de "aplicativo mobile" mesmo rodando no navegador do PC.

*Observação: não utilizei JavaScript, banco de dados ou backend. A navegação entre as telas é feita puramente com as tags `<a>` interligando os arquivos `.html`, e o redirecionamento dos formulários utiliza o atributo `action` para simular o avanço de telas (como a geração do protocolo).*

## Como rodar o projeto
É o jeito mais fácil possível, não precisa instalar nada!
1. Faça o download de todos os arquivos do projeto e coloque dentro de uma mesma pasta.
2. Dê dois cliques no arquivo `index.html`.
3. Ele vai abrir direto no seu navegador padrão (Chrome, Edge, Firefox, etc) e você já pode sair clicando nos botões.

##  O que eu aprendi
Foi um baita desafio estruturar tudo isso sem usar programação avançada e IA. Reaprendi muito sobre como usar o CSS para colocar os elementos direito, como organizar o código em arquivos separados e, principalmente, como planejar as telas pensando na jornada do usuário (Mockups e requisitos). 

A ideia é que no futuro esse projeto ganhe um banco de dados real e um backend para processar as denúncias de verdade!
